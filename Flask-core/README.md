# Core

## 파일 하나로 만드는 웹 애플리케이션

* hello.py 파일 하나만으로 웹 애플리케이션을 만든다.

## 애플리케이션과 실행 스크립트 분리

* run.py 실행 스크립트 파일과 hello.py 애플리케이션 파일을 분리한다.

## 애플리케이션을 패키지로 구성

* hello.py 애플리케이션 파일을 패키지로 구성한다.
* hello 폴더를 만들고 hello.py 파일의 내용이 그대로 __init__.py 파일이 된다.
