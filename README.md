# Flask 예제

## 목표

Flask 기초와 여러 가지 확장을 예제로 살펴본다.

주요 확장의 경우 개별 공식 문서의 내용을 기초로 하고 실제로 여러 가지 테스트를 통해 학습한 내용을 정리한다.
